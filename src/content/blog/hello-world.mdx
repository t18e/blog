# Hello World

This is an example blog post written in MDX. You can use all the power of Markdown and React components here.

## Features

- **Markdown Support**: Write your content in Markdown
- **Code Highlighting**: ```js
console.log('Hello World');
```
- **Custom Components**: Import and use React components
- **Frontmatter**: Add metadata to your posts

## Getting Started

1. Create a new `.mdx` file in the `src/content/blog` directory
2. Add frontmatter at the top of the file
3. Write your content in Markdown
4. Deploy!
